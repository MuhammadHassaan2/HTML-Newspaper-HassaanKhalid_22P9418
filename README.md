## Original Newspaper Concept

This project recreates a pre-1990 Urdu newspaper front page using only HTML.
The layout is inspired by traditional Pakistani newspaper formatting with
multiple columns, advertisement sections, headlines, and article blocks.

---

## Project Description

The goal of this assignment was to recreate a newspaper front page using
semantic HTML structure and table-based layout techniques. The design
imitates the dense column structure of older Urdu newspapers published
before 1990.

Inline styling was used to format borders, alignment, and spacing while
keeping the implementation limited to HTML only.

---

## Features Implemented

- HTML5 document structure
- Table-based 3-column newspaper layout
- Nested tables for small news sections
- Header masthead section
- Navigation strip
- Multiple article sections
- Timeline list
- Footer section
- Four images with alt attributes
- Inline styling for layout formatting
- Proper indentation and comments
